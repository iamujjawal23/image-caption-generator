# Image Caption Generator

This is an AI-powered Image Caption Generator built with **FastAPI**, **Deep Learning (TensorFlow/Keras)**, and a **Premium Custom Web Frontend**.

## Features
- **Upload Image**: Drag & Drop images to generate captions.
- **Micro-Animations**: Features a dynamic ambient canvas particle background and 3D hover effects.
- **FastAPI Backend**: Fast, lightweight handling of image processing and inference.

Click the button below to instantly deploy this project live on the internet using Render!

### Manual Instructions
1. Fork or push this repository to your GitHub.
2. Sign in to [Render](https://render.com).
3. Connect your GitHub and select this repository. it will automatically use the `render.yaml` to spin up the server!

*(Note to owner: Once deployed via Render, you can click the settings gear on your GitHub repository page and paste your new `.onrender.com` link into the "Website" URL field so everyone can find it!)*
